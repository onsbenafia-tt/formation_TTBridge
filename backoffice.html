Introduction
Le présent document constitue le manuel d’utilisation du système TTBridge BackOffice destiné aux Administrateurs DME. Cette solution permet la gestion des amicales et des conventions, ainsi que la consultation des adhérents et des demandes. Ce guide décrit les différentes fonctionnalités de l’interface web, notamment : L’authentification, la gestion des amicales, la gestion des conventions, ainsi que la consultation des adhérents et des demandes.

Authentification

Bienvenue dans TTBridge — la solution de gestion des amicales développée pour Tunisie Telecom.

L’utilisateur s’authentifie en saisissant son nom d’utilisateur ainsi que son mot de passe.

En cas d'échec, vérifiez votre connexion réseau et réessayez. Ne partagez jamais vos identifiants.
Les interfaces Web
1. Consultation des Amicales

Liste des Amicales

Cette interface permet de consulter, rechercher et gérer les amicales enregistrées dans le système.

  • Zone de recherche : permet de filtrer les amicales par nom via un mot clé, puis cliquer sur Appliquer.
  • Tableau des résultats :
    • Nom de l’amicale
    • Nombre d’adhérents
    • Nombre de conventions
    • Nombre de transactions
  • Actions : modifier ou supprimer une amicale.
  • Ajout : bouton Ajouter une amicale pour créer une nouvelle amicale.
  • Pagination : navigation entre les pages (Previous / Next).

1. Consultation des conventions

Liste des Conventions

Cette interface permet de consulter, filtrer et gérer les conventions.

  • Filtres de recherche :
    • Nom de la convention
    • Date de fin (intervalle)
    • Statut (Active / Expirée)
    • Responsable de paiement
  • Tableau des résultats :
    • Identifiant
    • Nom de la convention
    • Responsable paiement
    • Date de fin
    • Statut (affiché avec indicateur visuel)
  • Actions : consulter, modifier ou supprimer une convention.
  • Ajout : bouton Ajouter une convention.
  • Pagination : navigation entre les différentes pages.

2. Détails de la convention

Détails de la convention

On peut consulter les details d’une convention affichant les informations générales (nom, dates, statut).

Les paramètres comme nombre de produits, âge maximal et documents requis sont également visibles.

3. Ajout d'une convention

Ajouter une convention

On peut ajouter une convention en sélectionnant une amicale et en saisissant les informations générales (nom, dates, responsable paiement).

Les documents obligatoires ainsi que les différentes offres associées doivent être définis.

Cliquer sur Ajouter pour enregistrer ou Annuler pour quitter.

Liste des Adhérents

Cet écran permet de consulter et rechercher les adhérents associés aux amicales.

  • Zone de recherche : permet de rechercher un adhérent par CIN, CNRPS ou numéro de téléphone, puis cliquer sur Appliquer.

    • CIN
    • Prénom et Nom
    • Numéro de téléphone
    • Date de naissance
    • CNRPS
    • Région
    • Nom de l’amicale
  • Action : consulter les détails d’un adhérent via l’icône correspondante.
  • Pagination : navigation entre les pages des résultats.

1. Consultation des demandes

Liste des Demandes

Cet écran permet de consulter, rechercher et suivre les demandes (transactions) associées aux adhérents.

  • Zone de recherche : permet de rechercher une demande par :
    • ID de transaction
    • Informations de l’adhérent
    Cliquer sur Appliquer pour afficher les résultats filtrés.
  • Filtrage avancé :
    • Filtrer les demandes selon le statut (ex : Activée, En attente, Annulée)
  • Tableau des résultats :
    • ID : identifiant de la transaction (ex : TRAN-090)
    • Adhérent : nom de l’adhérent concerné
    • Statut :
      • En attente d’activation
      • Activée
      • Annulée
      • Informations supplémentaires (ex : Voucher livré / non livré)
  • Action : consulter les détails d’une demande via l’icône affichée.
  • Pagination : navigation entre les pages pour afficher plus de résultats.

2. Détails de la demande

Détails de la demande

Changer le mot de passe

Pour modifier le mot de passe :

  • Saisir le nouveau mot de passe
  • Confirmer le mot de passe dans le champ de confirmation

Une icône permet d’afficher ou de masquer le mot de passe pour vérifier la saisie.

Cliquer sur « Mettre à jour » pour enregistrer le nouveau mot de passe.

L’utilisateur peut également accéder à la fonctionnalité via le menu « Changer mot de passe » dans l’application.

En cas d’erreur ou si le mot de passe ne respecte pas les règles, un message d’alerte s’affiche.

Réinitialiser le mot de passe

En cas d’oubli du mot de passe, cliquer sur le lien « Mot de passe oublié ? » depuis la page de connexion.

Une page de réinitialisation s’affiche. L’utilisateur doit saisir son nom d’utilisateur ou son email.

Cliquer sur le bouton « Envoyer le lien » pour recevoir un lien de réinitialisation.

Après réception du lien et accès à la page dédiée, l’utilisateur est invité à définir un nouveau mot de passe.
