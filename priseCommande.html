Authentification

Connexion à l'application

Pour accéder à l’application TTBridge, l’utilisateur doit saisir ses identifiants de connexion.

Dans le champ Nom d’utilisateur, saisir l’identifiant ou l’adresse email fournie.

Dans le champ Mot de passe, saisir le mot de passe associé au compte.

Il est possible d’afficher ou de masquer le mot de passe en cliquant sur l’icône située dans le champ.

En cas d’oubli, cliquer sur le lien « Mot de passe oublié ? » pour lancer la procédure de réinitialisation.

Après saisie des informations, cliquer sur le bouton « Se connecter » pour accéder à l’application.

En cas d'échec, vérifiez votre connexion réseau et réessayez. Ne partagez jamais vos identifiants.

1. Recherche des demandes

Rechercher une demande

Cette page permet de rechercher des demandes en utilisant un mot clé.

Important : les demandes ne sont affichées qu’après saisie d’un critère de recherche.

Pour effectuer une recherche :

  • Saisir un mot clé (identifiant de transaction ou information de l’adhérent) dans le champ de recherche
  • Cliquer sur le bouton Appliquer

Après validation, la liste des demandes correspondantes s’affiche avec les informations principales : identifiant, adhérent, produit et statut.

Si aucun critère n’est renseigné ou si aucun résultat ne correspond à la recherche, le message « Aucune donnée » est affiché.

Une action permet de consulter le détail de chaque demande.

2. Détails de chaque demande

Consulter les détails et le suivi d’une demande

Pour consulter une demande, cliquer sur l’icône de visualisation correspondante dans la liste des demandes.

Une fenêtre s’ouvre affichant les informations détaillées réparties en trois sections :

  • Détails de la demande : identifiant, statut et source
  • Détails de l’adhérent : informations personnelles (CIN, nom, téléphone, région…)
  • Détails du produit : nom, subvention, montants et durée d’engagement

La section Suivi de la transaction permet de visualiser l’état d’avancement de la demande.

Les cas possibles sont les suivants :

  • Demande activée avec voucher livré : toutes les étapes sont complétées avec succès
  • Demande activée avec voucher non livré : possibilité de demander la livraison du voucher
  • Demande activée : demande sans voucher activée
  • Demande annulée : la demande est rejetée, l’étape d’activation est marquée comme annulée
  • Demande En attente confirmation amicale : la demande est en attente de confirmation par l'amicale
  • Demande En attente activation :la demande est en attente le lancement de l’activation

Selon le statut de la demande, certaines actions peuvent être disponibles :

  • Renvoyer SMS : permet de renvoyer les informations à l’adhérent
  • Demander livraison voucher : activé uniquement si le voucher n’est pas encore livré

Cliquer sur l’icône de fermeture pour quitter la fenêtre de détails.

Traiter une demande en attente d’activation

Pour traiter une demande en attente d’activation, cliquer sur l’icône de consultation correspondante dans la liste des demandes.

Une fenêtre s’ouvre affichant les informations détaillées de la demande, de l’adhérent et du produit.

Le statut de la demande est « En attente d’activation », ce qui signifie que l’activation n’a pas encore été réalisée.

Pour procéder à l’activation, l’utilisateur doit renseigner le numéro MSISDN (numéro de téléphone).

Étapes à suivre :

  • Saisir le numéro dans le champ Numéro (MSISDN)
  • Cliquer sur le bouton Tester numéro

Après le test, deux cas peuvent se présenter :

  • Numéro valide : un message de validation s’affiche. Le bouton Activer devient disponible pour finaliser l’activation.
  • Numéro invalide : un message d’erreur s’affiche indiquant que le numéro n’est pas associé à l’adhérent ou que la convention est incompatible.

Une fois le numéro validé, cliquer sur Activer pour activer la demande.

D’autres actions sont également disponibles :

  • Renvoyer SMS : permet d’envoyer à nouveau les informations à l’adhérent
  • Télécharger : permet de récupérer les informations ou documents liés à la demande

Après activation, la demande passe au statut Activée et le suivi de la transaction est mis à jour.

1. Nouvelle Demande

Pour créer une demande, accéder au menu « Nouvelle Demande ».

L’utilisateur peut rechercher un adhérent en saisissant le CIN ou le CNRPS.

Deux cas peuvent se présenter :

  • Adhérent trouvé : les informations de l’adhérent s’affichent automatiquement.
  • Aucun adhérent trouvé : un message s’affiche et l’utilisateur peut ajouter manuellement un adhérent ou importer plusieurs adhérents via Excel.

Une fois l’adhérent sélectionné, procéder aux étapes suivantes :

  • Sélectionner une amicale
  • Sélectionner une convention
  • Choisir le produit correspondant

Un champ facultatif permet de renseigner le numéro d’appel (MSISDN).

Il est également possible :

  • Cliquer sur Ajouter adhérent pour créer un nouvel adhérent
  • Cliquer sur Importer plusieurs adhérents pour effectuer un ajout en masse

Dans certains cas, il est nécessaire d’importer des documents (ex : CIN, attestation de travail) en cliquant sur le bouton Upload PDF.

Une fois toutes les informations renseignées et les documents ajoutés, cliquer sur le bouton Demander pour valider la création de la demande.

La demande est ensuite enregistrée avec le statut « En attente » jusqu’à son traitement.

Changer le mot de passe

Pour modifier le mot de passe :

  • Saisir le nouveau mot de passe
  • Confirmer le mot de passe dans le champ de confirmation

Une icône permet d’afficher ou de masquer le mot de passe pour vérifier la saisie.

Cliquer sur « Mettre à jour » pour enregistrer le nouveau mot de passe.

L’utilisateur peut également accéder à la fonctionnalité via le menu « Changer mot de passe » dans l’application.

En cas d’erreur ou si le mot de passe ne respecte pas les règles, un message d’alerte s’affiche.

Réinitialiser le mot de passe

En cas d’oubli du mot de passe, cliquer sur le lien « Mot de passe oublié ? » depuis la page de connexion.

Une page de réinitialisation s’affiche. L’utilisateur doit saisir son nom d’utilisateur ou son email.

Cliquer sur le bouton « Envoyer le lien » pour recevoir un lien de réinitialisation.

Après réception du lien et accès à la page dédiée, l’utilisateur est invité à définir un nouveau mot de passe.
